# is_windows_lib

## Proto type

```c
int is_windows_lib (void)
```

## Description

운영체제가 windows machine 인지 판단을 한다. 운영체제가 windows 이면 1 을 리턴하고 아니며 0을 리턴한다.

## Arguments

void

## Example

```php
<?php
if ( is_windows_lib () )
  echo "This is windows machine\n";
else
  echo "This is not windows machien\n";
?>
```
